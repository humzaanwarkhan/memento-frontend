# Memento : Secure Memory & Notes Manager

Memento is a privacy-focused personal memory & notes management application designed to protect user data through strong authentication mechanisms and secure backend practices.

WEBSITE LIVE AT: https://memento-frontend-phi.vercel.app/

## Why Memento is Secure

Memento is not just another notes app â€” it is built with a **security-first mindset**.

- Enforced authentication & authorization on every API endpoint  
- Token-based access control to prevent unauthorized data access  
- Input validation & sanitization to prevent injection attacks  
- Designed to minimize exposure of sensitive user information  
- Planned encrypted storage strategy for future releases  

## Tech Stack

- Backend: Node.js, Express  
- Database: MongoDB  
- Authentication: JWT Authentication   
- Deployment: Render (backend), Vercel (frontend)

## Features

- Secure user registration & login  
- Protected routes for private data  
- Password reset flow with email verification  
- Clean modular backend architecture  
- Security-oriented design principles  

## Purpose of the Project

Memento was built to:

- Practice writing **secure backend code**  
- Understand authentication vulnerabilities  
- Simulate real-world production security flows  
- Let users write journals with security and fun
